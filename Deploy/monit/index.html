# Monit 簡介及建置教程

# Monit 簡介 官網

Monit 是一個常用來管理監控 Linux 系統的小工具，經過簡單的設定就可以在系統異常時採取行動，包括警報、重啟、執行腳本等等。Monit 能夠進行自動維護，修復，並在錯誤情況下運行有意義的因果操作。M/Monit 則是與其配套的網頁監控服務。

# 指令

設定檔檢測:
monit -t
重新加載:
monit reload
開啟:
service monit start
關閉:
service monit stop
重啟:
service monit restart
