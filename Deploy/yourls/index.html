# 私人短網址服務 Yourls 部屬教程

# 簡介

短網址服務用途廣泛，不論是報告、引用資料、分享文件等，然而不同服務商提供的服務穩定度也不同，如 google 在 2018 年停止提供短網址服務，Yourls 是一套以 php 寫成的輕量開源短網址服務，簡易配置且具有管理介面，可以自定義別名、期限等等，架設自己的短網址服務，本教程的目的就是以 Docker 及 Yourls 部屬自己的短網址服務。
